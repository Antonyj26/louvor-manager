# 🎸 Louvor Manager

Um sistema moderno de gestão para ministérios de louvor, focado em organização de escalas, repertórios e integração de equipes.

## 🚀 O Projeto
O **Louvor Manager** está sendo desenvolvido para facilitar o dia a dia de líderes de louvor. Este repositório contém o Front-end construído com uma interface moderna, responsiva e focada na experiência do usuário (UX).

## 🛠️ Tecnologias Utilizadas
* **React 19** (Vite)
* **TypeScript**
* **Tailwind CSS v4**
* **React Router Dom 7**
